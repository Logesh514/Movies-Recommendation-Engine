# Movie Recommendation Engine

## Objective
To recommend similar movies based on genre using Machine Learning techniques.

## Tools and Technologies
- Python
- Pandas
- Scikit-Learn
- Google Colab

## Dataset
Sample Movie Dataset

## Algorithm Used
Cosine Similarity

## Features Used
- Movie Genre

## Project Workflow
1. Create movie dataset
2. Convert genres into numerical features
3. Calculate similarity between movies
4. Recommend similar movies

## Result
The system successfully recommended movies based on genre similarity.

## Example Output
Input Movie: Avengers

Recommended Movies:
- Iron Man
- Endgame
- Avatar
